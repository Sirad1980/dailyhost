Daily Host Generator - Your Scrum Team's Companion!

Description:

Daily Host Generator, developed by Sirad Salkanovic, is a powerful Chrome extension designed to streamline the process of determining daily hosts for your scrum meetings. This handy tool empowers scrum teams by providing the flexibility to generate hosts for both today and tomorrow, ensuring efficient planning and organization.

Key Features:

  • Generate Daily Hosts: Easily generate hosts for today's and tomorrow's scrum meetings with just a click. Simplify the planning process and ensure smooth daily stand-ups for your team.
  • Skip Host Option: Unforeseen circumstances? No problem! Use the skip host feature to adjust your schedule seamlessly, keeping your scrum meetings flexible and accommodating.
  • GitHub Integration: Daily Host Generator stores data in a GitHub file, ensuring secure and accessible storage for your scrum team's daily hosts. To get started, you'll need a GitHub account and a personal access token.
  • Customizable Settings: Tailor the extension to fit your team's unique needs with the settings dialog. Enter your GitHub repository details, username, file path, and access token for a personalized experience.

How to Use:

  1. Install the Daily Host Generator Chrome extension.
  2. Open the settings dialog and input your GitHub repository details, username, file path, and access token.
  3. Generate hosts effortlessly for today and tomorrow or skip hosts as needed.

Why Daily Host Generator?

  • Save Time: Streamline the process of determining hosts for daily scrum meetings.
  • Flexibility: Easily skip hosts to accommodate unexpected changes in your team's schedule.
  • GitHub Integration: Securely store and access your data in a GitHub file for enhanced collaboration.

Note: A GitHub account and a personal access token are necessary for using the Daily Host Generator. Ensure you have the required credentials to unlock the full potential of this extension.

Enhance your scrum team's daily stand-ups with Daily Host Generator - because effective communication starts with organized hosting!
